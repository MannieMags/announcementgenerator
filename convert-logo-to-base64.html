Step 4: Update script.js

After copying the base64 code:

  1. Open script.js in a text editor
  2. Find the line: defaultLogo.src = 'SADVLOGO.png'; (around line 41)
  3. Replace it with: defaultLogo.src = 'YOUR_BASE64_CODE_HERE';
  4. Paste the copied base64 code in place of YOUR_BASE64_CODE_HERE
  5. Save the file and refresh your browser
